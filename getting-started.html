Getting Started

Setting up your first project.

1. Install Node on Fire

Install Node on Fire (globally):

$ npm install -g fire

Set up the initial files and folders of the app:

$ fire apps:create helloworld

You can now run the app:

$ cd helloworld/

$ fire run

To view the app, head over to http://127.0.0.1:5000/. When you make changes, run fire build && fire release and then fire run.

Configure the database

If you have Postgres running locally, you can quickly create a new database and update your environment variable:

$ fire datastore:create helloworld

What's next?

Have a look at the example projects or browse through the documentation. If you have any questions, please reach out on Twitter. Enjoy!
