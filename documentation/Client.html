Class: Client

Client

new Client(internal, done)

A client is a connection to the datastore, see Datastore#connect, taken from the pg's connection pool.

Parameters:
Name Type Description
internal pg.Client

The client from the pg module.
done function

When invoked, gives the client back to the connection pool.

Methods

begin()

Begins a database transaction.

Please make sure to invoke Client#commit to commit a transaction or Client#rollback to cancel a transaction.

Returns:

Promise resolves with this instance.

Type
Promise

commit()

Commits a database transaction.

See Client#begin.

Returns:
Type
Promise

query(query, parameters)

Wrapper function around pg.Client#query to return a Promise instead.

Parameters:
Name Type Description
query String

The SQL statement to execute.
parameters Array

Any parameters passed to SQL statement. The first item is replaced with $1 in the query, etc.
Returns:

Returns a result set, see pg.Client#query for the specifics.

Type
Promise

rollback()

Rolls back a transaction.

See Client#begin.
