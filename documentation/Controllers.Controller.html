Class: Controller

Controller

new Controller(models, request, response)

The base constructor for a controller. Inheritance is applied when calling App#controller on a controller constructor.

To prevent leaking of data between requests, for every request a new instance is allocated and initialized. This is likely to change in the future because of optimizations.

The base path or prefix of every route in this controller. For example, a basePathComponents of ['api', 'v1'] would prefix every route with /api/v1.

The models module. On the models module you can access the models. For more information see Models.

Cookie-based session storage, see https://github.com/expressjs/cookie-session.
