Class: MigrationTask

MigrationTask

new MigrationTask(name, model, params, method)

A migration consists of several migration tasks. A migration task is, for example, creating a model or editing a model.

Migration tasks store information to persist models to the database.

Parameters:
Name Type Description
name String

The name of the migration task.
model Model

The model associated to the task. Passed to method.
params Dictionary

Task-specific parameters passed to method when executed.
method function

The function to invoke once the task is executed.

Methods

execute()

Executes the migration task's method with model and params as arguments.

Returns:
Type
Mixed
