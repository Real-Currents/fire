__parseRow(row)

Parses the row and returns the associations and the column map parsed.

Parameters:
Name Type Description
row Dictionary

A single row from the datastore.
Returns:

Multiple return values in an array. At index 0 the columns found, and at index 1 the associations found. The array contains 2 items.

Type
Array

parseAuthorization(authorization)

Parses the authorization header and returns an array with two values: the user name, the password.

This function either returns null, or an array with always a length of 2.

Parameters:
Name Type Description
authorization String

The basic auth value. The word "Basic " followed by a base64 encoded username and password combination concatenated by a colon character.
