Building your first app

Let's create a simple survey for our users.

Before You Start

Do you have node and npm installed? If not, head over to nodejs.org to download and install the latest version.

Please make sure you have Node on Fire installed, ideally globally. You can install Node on Fire by running:

$ npm install -g fire

Create the Project

Now we'll create the project. Open a terminal window and create a new project (we'll call it Survey Chef \o/):

$ fire apps:create surveychef

When running this command Node on Fire will create a new folder and install the initial project's dependencies.

Are you running in any errors at this point? Please reach out on Twitter and we'll be able to help you out.

Configure the Database

We will also be using a PostgreSQL datastore. Please configure a database locally, or set-up a free Heroku database.

If you have Postgres running locally, you can quickly create a new database and update your environment variable:

$ fire datastore:create surveychef

This creates a new database and stores the connection string in the .env file. If you are using a remote Heroku datastore, please paste the connection string in your .env file like below:

DATABASE_URL=postgres://user:password@hostname:port/database?ssl=true

Please note the last part in the connection string. You should make sure ?ssl=true is added to your connection string.
