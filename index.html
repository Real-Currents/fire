Node on Fire

A JavaScript-everywhere, model-driven, code generation web framework built on top of AngularJS, express and PostgreSQL.

Welcome to Node on Fire

Node on Fire is a JavaScript-everywhere, model-driven, code-generation web framework. In Node on Fire, your models are available on both the client-side and server-side and your controllers run in both the client- and server-side.

JavaScript Everywhere

You write your complete app in JavaScript*, both the client- and server-side. You don't just put server code in /server/* and client code in /client/*, no, your controllers and models are truly available everywhere.

* If you prefer, you can also write you app in CoffeeScript, of course.

Scaffold Everything

Node on Fire loves Yeoman. A generator is available so you can scaffold your models, controllers, directives, or even the example apps. Install the generator via npm install -g generator-fire and run yo fire to get started with the generator.

Not familiar with Yeoman? Check it out.

AngularJS

Node on Fire completely embraces the superheoric AngularJS framework for all things client-side.

Model-Driven

Configure your models, and Node on Fire generates migrations, database schema, a RESTful API and a client—server equal models API.
