## Node on Fire :fire:
A JavaScript-everywhere, model-driven, web framework built on top of [AngularJS](https://angularjs.org/), [express](http://expressjs.com/) and [PostgreSQL](http://www.postgresql.org/).

### Installation
```
npm install -g fire
```

### Documentation

For additional reading and documentation please visit http://nodeonfire.org/.

### Help
